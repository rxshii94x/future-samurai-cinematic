## 📖 Overview

Future Samurai Cinematic Experience is a single page immersive web application designed to showcase advanced frontend engineering and cinematic user experiences.

The application combines Japanese artistic inspiration with futuristic science fiction to create a unique digital journey through multiple interactive chapters.

Every section is designed to feel like a scene from a high budget cinematic production while maintaining smooth performance across desktop and mobile devices.

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/rxshii94x/future-samurai-cinematic.git
```

Go to the project

```bash
cd future-samurai-cinematic
```

Install dependencies

```bash
npm install
```

Start development server

```bash
npm run dev
```

Create production build

```bash
npm run build
```

Preview production build

```bash
npm run preview
```
